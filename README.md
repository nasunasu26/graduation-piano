# graduation-piano
RUNTEQ 卒業制作用 ピアノWebアプリ

## サービス概要
本サービスは、ブラウザ上で気軽にピアノ演奏を楽しめるWebアプリです。
画面上の鍵盤をクリックすることで音を鳴らし、自由に演奏できます。
また、簡単なメロディーを覚えて演奏する練習モードも備えています。

## このサービスへの思い・作りたい理由
小さい頃からピアノに憧れはあったものの、
ピアノ本体の価格が高いことやレッスン代がかかることから、
始めるまでのハードルが高いと感じていました。

また、実際に始めたとしても
「継続して練習しなければならない」という心理的なプレッシャーがあり、
気軽に挑戦することが難しいと感じる場面が多くありました。

そこで、楽器やレッスンがなくても、
思い立ったときにすぐ音を鳴らせる、
練習という意識を持たずに触れられるピアノ体験を提供したいと考え、
本アプリの開発を行いました。

## ユーザー層について
ピアノや音楽に興味はあるが、未経験・初心者の人
子ども向けの簡単な音遊びアプリを探している人
ゲーム感覚で音感や記憶力を鍛えたい人
専門的な知識がなくても使えることを重視し、誰でも直感的に操作できるユーザー層を想定しています。

## サービスの利用イメージ
ユーザーはサイトにアクセスすると、すぐにピアノ鍵盤画面へ進むことができます。
自由演奏モードでは、鍵盤をクリックするだけで音が鳴り、好きなように演奏を楽しめます。
また、練習モードではお手本のメロディーを聴いたあとに同じメロディーを演奏することで、ゲーム感覚で練習ができます。

## ユーザーの獲得について
SNSやポートフォリオサイトでの公開を想定しています。
「ブラウザで遊べるピアノアプリ」として気軽に試してもらい、音楽に触れる入口として利用してもらうことを想定しています。

## サービスの差別化ポイント・推しポイント
本サービスは、既存の高機能なピアノアプリとは異なり、
「継続的な練習」や「上達」を前提としない点を特徴としています。

多くのピアノアプリは、
・継続利用
・練習の積み重ね
・正確な演奏
を前提とした設計になっており、
初心者にとっては心理的なハードルが高くなりがちです。

本サービスでは、
心理的ハードルを下げ、
**継続しなくてもよく、思いついたときにいつでもさっと触れられる**
ピアノ体験を重視しています。

ログインや設定を必要とせず、
画面を開けばすぐに音が鳴ることで、
「練習するぞ」という構えを作らずに、
音楽に触れる入口となることを目指しています。

上達するためのツールではなく、
音を鳴らすことそのものを楽しめることを
本サービスの強みとしています。

## 機能拡張について
今後のアップデートでは、
以下のような機能追加を検討しています。

- ピアノ以外の音色（ギターなど）への切り替え
- 練習曲の追加
- 曲の作者名による検索機能

音色や楽曲の幅を広げることで、
ユーザーが飽きずに音楽に触れ続けられる体験を目指しています。

### テーブル詳細
#### songs テーブル
- id : bigint（PK）
- title : string（曲名）
- notes : text（曲の音階データ）
- created_at : datetime
- updated_at : datetime

#### play_histories テーブル
- id : bigint（PK）
- song_id : bigint（FK / songs.id）
- correct : boolean（正解したかどうか）
- played_at : datetime（演奏した日時）
- created_at : datetime
- updated_at : datetime
