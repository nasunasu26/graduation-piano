# graduation-piano
RUNTEQ 卒業制作用 ピアノWebアプリ

## サービス概要
本サービスは、ブラウザ上で気軽にピアノ演奏を楽しめるWebアプリです。
画面上の鍵盤をクリックすることで音を鳴らし、自由に演奏できます。
また、簡単なメロディーを覚えて演奏する練習モードも備えています。

## このサービスへの思い・作りたい理由
小さい頃からピアノに憧れはあったものの、ピアノ本体の価格やレッスン代の問題から気軽に始めることが難しいと感じていました。

また、継続して練習しなければならないプレッシャーもあり、
もっと「気軽に触れられるピアノ体験があればいい」と考え本アプリを制作しています。

## ユーザー層について
- ピアノや音楽が好きだが初心者の人
- 子ども向けの音遊びアプリを探している人
- ゲーム感覚で音感や記憶力を鍛えたい人

## サービスの利用イメージ
アクセスするとすぐにピアノ画面へ進め、自由演奏モードで好きなように音を鳴らせます。
練習モードでは「きらきら星」を題材に、正しく演奏できるまで挑戦できます。

## ユーザーの獲得について
SNSやポートフォリオサイトで公開し、初心者が音楽に触れるきっかけになることを狙います。

## サービスの差別化ポイント
- 練習・継続を前提としない「気軽に遊べるピアノ体験」
- 楽譜や専門知識が不要
- ログイン不要で即プレイ可能

## 機能候補
### MVPリリースで実装する機能
- ピアノ鍵盤の表示
- 鍵盤クリックによる音の再生
- 自由演奏モード
- メロディー練習モード（きらきら星）

### 本リリースまでに追加したい機能
- 練習曲の追加
- 正解・不正解の演出強化
- スマートフォン操作性の向上

### 将来的な機能拡張
- 音色切り替え
- 楽曲検索機能 など

## テーブル詳細

### songs テーブル
- id : bigint（PK）
- title : string（曲名）
- notes : text（音階データ）
- created_at : datetime
- updated_at : datetime

### play_histories テーブル
- id : bigint（PK）
- song_id : bigint（FK / songs.id）
- correct : boolean（正解かどうか）
- played_at : datetime（演奏日時）
- created_at : datetime
- updated_at : datetime
